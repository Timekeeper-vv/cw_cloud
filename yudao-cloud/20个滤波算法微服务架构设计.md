# 🎯 20个滤波算法微服务架构设计

> 基于老师录音要求和工业健康监测技术

**版本**: v1.0
**更新时间**: 2025-10-14 00:45

---

## 📋 项目背景（来自录音）

### 老师的核心需求

1. **20个微服务架构**
   - 每个微服务实现一个滤波算法（FX函数）
   - 编号1-20，如：卡尔曼滤波、LMS、NLMS、均值滤波等
   - 使用Docker容器化部署
   - 通过Kubernetes编排管理

2. **实时性要求**
   - 数据采集→微服务处理→报警要快速响应
   - 实时调用滤波微服务进行信号处理
   - 立即发现异常并报警

3. **应用场景**
   - 航天领域：发动机健康监测
   - 航空领域：飞机部件故障检测
   - 桥梁检测：结构断裂监测
   - 文物保护：表面裂缝监测
   - 工业制造：设备故障预警

4. **技术架构**
   - Spring Boot微服务
   - 声发射传感器数据采集
   - Web平台可视化展示
   - 数据存储到MySQL
   - REST接口调用微服务
