# Signal-2 数据说明

## 📊 数据信息

包含三个独立的 TDMS 文件：

### 文件列表

1. **ae_sine_2s.tdms**
   - 纯净的正弦波信号
   - 频率: 5 kHz
   - 时长: 2 秒
   - 采样率: 100 kHz

2. **ae_noise_2s.tdms**
   - 纯噪声信号
   - 宽频噪声
   - 时长: 2 秒
   - 采样率: 100 kHz

3. **ae_mix_2s.tdms**
   - 混合信号 (正弦波 + 噪声)
   - 包含 5 kHz 信号和噪声
   - 时长: 2 秒
   - 采样率: 100 kHz

## 🔧 使用方法

### 运行信号滤波可视化脚本 V2

```bash
cd e:\Code\CW_Cloud\floatdata\floatdata-streaming
python signal-filter-visualizer-v2.py
```

### 输出结果

脚本会生成两张图片：

1. **complete_signal_analysis.png** - 完整时域分析（5个子图）
   - 子图1: 原始正弦波
   - 子图2: 纯噪声
   - 子图3: 混合信号（加噪后）
   - 子图4: 滤波后信号
   - 子图5: 原始 vs 滤波后（叠加对比）

2. **frequency_analysis.png** - 频域分析（4个子图）
   - 子图1: 原始正弦波频谱
   - 子图2: 噪声频谱
   - 子图3: 混合信号频谱
   - 子图4: 滤波后频谱

### 性能指标

脚本会输出：
- 滤波前后 MSE 对比
- 相关系数提升
- 滤波改善百分比

## 📝 数据特点

- **分离式数据**: 原始信号、噪声、混合信号各自独立
- **便于分析**: 可以清晰看到噪声如何影响信号
- **滤波验证**: 可以直观评估滤波器性能
