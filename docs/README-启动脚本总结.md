## 🔍 故障排查

### 问题1: "演示模式，无法进行写操作"
**解决**: 
- 使用 `START.bat` 启动（已自动关闭演示模式）
- 或运行 `restart-with-no-demo.bat`（如编码已修复）

### 问题2: 批处理脚本显示乱码
**解决**:
```powershell
# 运行编码修复脚本
powershell.exe -ExecutionPolicy Bypass -File fix-bat-encoding.ps1
```

### 问题3: 菜单不显示
**解决**:
1. 运行 `import-all-menus.bat` 导入菜单
2. 清除浏览器缓存
3. 退出登录，重新登录

### 问题4: 服务启动失败
**检查**:
- MySQL是否运行（端口3306）
- Redis是否运行（端口6379）
- 端口是否被占用
- JAR文件是否存在（target目录下）
